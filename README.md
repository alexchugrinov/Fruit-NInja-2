# Project D

A lightweight HTML / JavaScript starter workspace.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm start
   ```
3. Open `index.html` in the browser if the server does not start automatically.

## Workspace files

- `index.html` – app entry point
- `styles.css` – basic styling
- `app.js` – JavaScript demo behavior

## Recommended VS Code extensions

- Live Server
- Prettier
- ESLint
